## Description
2-player Snakes and ladders, but online. 

## Tech Stack
- Python (tkinter, turtle, microbit)
- micro:bit
- Thonny (code editor)

## Get started
1. Clone repo / Download files
2. Get a micro:bit
3. Download the correct MicroPython game firmware from [here](https://microbit.org/get-started/user-guide/firmware/)
4. Connect the micro:bit and flash it with the .hex file.
5. Run `03 Microbit Game.py`
6. Take turns and play :)

## How to play
1. Choose Player A (left) vs B (right)
2. Shake micro:bit when it's your turn
3. Press the button (on your side) for the number that you rolled
4. First person to reach the end wins
